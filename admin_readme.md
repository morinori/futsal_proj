# 📋 SGSC 풋살팀 관리시스템 - 관리자 매뉴얼

## 🔐 관리자 로그인 정보

### 기본 계정 정보
| 사용자명 | 비밀번호 | 이름 | 역할 |
|---------|---------|------|------|
| `admin` | `admin123` | 시스템 관리자 | 시스템 전체 관리 |
| `captain` | `captain123` | 김팀장 | 팀 운영 관리 |
| `manager` | `manager123` | 이총무 | 재정 및 행정 관리 |

### 로그인 방법
1. 웹사이트 접속: `http://localhost:8501`
2. 사이드바 하단 **👤 관리자 로그인** 클릭
3. 사용자명과 비밀번호 입력 후 **로그인** 클릭

---

## 📋 관리자 전용 기능

### 🔐 인증이 필요한 페이지
관리자 로그인 후에만 접근 가능한 기능들:

- **📅 일정 관리**: 경기 일정 생성, 수정, 삭제
- **👥 선수 관리**: 선수 등록, 정보 수정, 삭제
- **💰 재정 관리**: 팀 재정 기록 및 분석
- **⚙️ 관리자 설정**: 관리자 계정 관리

---

## 📅 일정 관리 기능

### 경기 일정 등록
1. **📅 일정 관리** 페이지 접속
2. **➕ 경기 추가** 탭 선택
3. 필수 정보 입력:
   - 구장 선택 (미리 등록된 구장에서 선택)
   - 경기 날짜 및 시간
   - 상대팀 이름
   - 경기 결과 (경기 후 업데이트 가능)
4. **경기 추가** 버튼 클릭

### 경기 일정 관리
- **📋 경기 목록** 탭에서 등록된 경기 확인
- 경기별 상세 정보 조회 및 수정
- 경기 삭제 (필요 시)

### 구장 관리
- **🏟️ 구장 관리** 탭에서 구장 정보 관리
- 새 구장 등록, 기존 구장 정보 수정

---

## 👥 선수 관리 기능

### 선수 등록
1. **👥 선수 관리** 페이지 접속
2. **➕ 선수 추가** 탭 선택
3. 필수 정보 입력:
   - 이름 (중복 불가)
   - 포지션 (GK/DF/MF/FW)
   - 전화번호 (선택사항)
   - 이메일 (선택사항)
4. **선수 추가** 버튼 클릭

### 선수 정보 수정
1. **👥 선수 목록** 탭에서 수정할 선수 찾기
2. **✏️ 수정** 버튼 클릭
3. 정보 수정 후 **수정 저장** 클릭

### 선수 삭제
1. **👥 선수 목록**에서 삭제할 선수 선택
2. **🗑️ 삭제** 버튼 클릭
3. 확인 메시지 후 다시 **🗑️ 삭제** 클릭하여 완전 삭제

### 출석 관리
- **📋 출석 관리** 탭에서 경기별 출석 현황 관리
- 선수별 출석/불참/미정 상태 업데이트
- 출석률 통계 확인

---

## 💰 재정 관리 기능

### 재정 기록 추가
1. **💰 재정 관리** 페이지 접속
2. **📝 기록 추가** 탭 선택
3. 거래 정보 입력:
   - 날짜
   - 설명 (용도 상세 기재)
   - 금액
   - 유형 (수입/지출)
   - 카테고리 (경기/회비/장비/행사/기타)
4. **💰 기록 추가** 버튼 클릭

### 재정 현황 확인
- **📊 재정 현황** 탭에서 전체 재정 요약 확인
- 총 수입, 총 지출, 현재 잔고 표시
- 월별 수입/지출 차트 분석
- 카테고리별 지출 분포 확인

### 거래 내역 관리
- **📋 거래 내역** 탭에서 모든 거래 기록 조회
- 필터링 및 정렬 기능 제공
- **🗑️ 삭제** → **⚠️ 확인** 버튼으로 잘못된 기록 삭제

### 재정 분석
- **📈 분석** 탭에서 상세 재정 분석
- 월별 수지 추이 그래프
- 지출 패턴 분석
- 재정 건전성 진단 및 개선 제안

---

## ⚙️ 관리자 설정

### 새 관리자 계정 생성
1. **⚙️ 관리자 설정** 페이지 접속
2. **➕ 계정 추가** 섹션 이용
3. 계정 정보 입력:
   - 사용자명 (중복 불가)
   - 비밀번호 (8자 이상)
   - 이름
   - 역할 (선택사항)
4. **관리자 추가** 버튼 클릭

### 기존 계정 관리
- 관리자 계정 목록 확인
- 계정 비활성화 기능
- 비밀번호 변경 기능

### 보안 관리
- **자동 로그아웃**: 비활성 시간 후 자동 로그아웃
- **세션 관리**: 안전한 세션 상태 관리
- **권한 제어**: 관리자 전용 기능 접근 제어

---

## 🚨 보안 주의사항

### 비밀번호 관리
- **정기적 변경**: 3개월마다 비밀번호 변경 권장
- **강력한 비밀번호**: 8자 이상, 숫자/영문/특수문자 조합
- **공유 금지**: 관리자 계정 정보 타인과 공유 절대 금지

### 접근 제한
- **공용 컴퓨터 사용 금지**: 개인 기기에서만 관리자 기능 사용
- **로그아웃 필수**: 사용 후 반드시 로그아웃
- **세션 타임아웃**: 30분 비활성 시 자동 로그아웃

### 데이터 관리
- **백업**: 중요한 재정 기록은 별도 백업 권장
- **삭제 주의**: 선수/거래 삭제는 복구 불가능
- **정확한 입력**: 재정 기록은 정확한 정보 입력 필수

## 🛠️ 문제 해결

### 자주 발생하는 문제

**Q: 로그인이 되지 않아요**
- A: 사용자명/비밀번호 정확히 입력했는지 확인
- A: 계정이 비활성화되지 않았는지 확인
- A: 브라우저 쿠키/캐시 삭제 후 재시도

**Q: 페이지가 로드되지 않아요**
- A: 새로고침(F5) 후 재시도
- A: 브라우저 개발자 도구에서 오류 메시지 확인
- A: 다른 브라우저에서 접속 시도

**Q: 데이터가 저장되지 않아요**
- A: 필수 입력 항목이 모두 작성되었는지 확인
- A: 인터넷 연결 상태 확인
- A: 세션 타임아웃 여부 확인 후 재로그인

**Q: 모바일에서 캘린더가 이상해요**
- A: 터치로 캘린더 조작 시 월 뷰가 유지되도록 최적화됨
- A: 경기 일정을 터치하여 출석 관리로 이동

### 긴급 상황 대처

**데이터 손실 시:**
1. 즉시 시스템 사용 중단
2. 백업 파일 확인
3. 시스템 관리자에게 연락

**보안 침해 의심 시:**
1. 즉시 모든 관리자 계정 비밀번호 변경
2. 의심스러운 활동 기록 확인
3. 필요시 새로운 관리자 계정 생성 후 기존 계정 비활성화

## 📈 시스템 활용 팁

### 효율적인 팀 관리
1. **정기적인 업데이트**: 주 1회 이상 시스템 정보 업데이트
2. **데이터 분석 활용**: 월간 재정 분석으로 예산 계획 수립
3. **출석률 관리**: 선수별 출석률 모니터링으로 팀 참여도 향상
4. **캘린더 활용**: 경기 일정을 캘린더로 시각적 관리

### 업무 분담 권장
- **팀장(Captain)**: 경기 일정, 선수 관리 담당
- **총무(Manager)**: 재정 관리, 출석 관리 담당
- **시스템 관리자(Admin)**: 전체 시스템 관리, 계정 관리 담당
