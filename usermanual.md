## 📋 출석 관리

출석 관리는 팀원들이 가장 많이 사용하는 기능입니다.

### 📅 경기별 출석

**경기 선택하기**
1. 좌측 사이드바에서 "📋 출석 관리" 클릭
2. "📅 경기별 출석" 탭 선택
3. 드롭다운에서 원하는 경기 선택

**출석 현황 보기**
- ✅ **참석**: 확정적으로 참석하는 선수
- ❌ **불참**: 참석이 어려운 선수
- ❓ **미정**: 아직 결정하지 않은 선수

**출석률 확인**
- 상단에 전체 출석 통계 표시
- 참석률 진행바로 한눈에 파악
- 실시간 업데이트

### 👤 개인별 출석 관리

**"누구세요?" 기능**
1. "👤 개인별 출석" 탭 클릭
2. "👤 누구세요?" 드롭다운에서 본인 이름 선택
3. 참석할 경기를 선택

**출석 상태 변경하기**
1. 원하는 경기 선택
2. 현재 상태 확인
3. 상태 변경 버튼 클릭:
   - **✅ 참석**: 확실히 참석할 수 있음
   - **❌ 불참**: 참석이 어려움
   - **❓ 미정**: 아직 확실하지 않음

**💡 팁**
- 상태 변경 시 즉시 팀원들에게 반영됩니다
- 마음이 바뀌면 언제든 다시 변경 가능합니다
- 경기 당일까지 변경할 수 있습니다

## 📱 모바일 사용법

### 모바일 최적화 기능

**터치 조작**
- 모든 버튼이 터치에 최적화되어 있습니다
- 달력의 경기를 터치하면 출석 관리로 이동합니다
- 스와이프로 페이지 간 이동 가능

**화면 크기 자동 조절**
- 화면 크기에 맞게 자동으로 레이아웃 조정
- 작은 화면에서도 모든 기능 사용 가능
- 세로/가로 방향 모두 지원

**빠른 출석 체크**
1. 홈화면에서 바로 다음 경기 확인
2. "출석 관리" 버튼으로 빠른 이동
3. 간편한 터치로 출석 상태 변경

### 모바일 사용 팁
- **와이파이 연결**: 데이터 사용량 절약을 위해 와이파이 사용 권장
- **북마크 추가**: 홈화면에 바로가기 추가하면 편리합니다
- **알림 허용**: 브라우저 알림을 허용하면 중요한 소식을 놓치지 않습니다

## 🤔 자주 묻는 질문

### Q1: 출석 상태를 변경했는데 반영이 안 돼요
**A:** 다음 단계를 따라해 보세요:
1. 페이지 새로고침 (F5 또는 브라우저 새로고침)
2. 올바른 경기를 선택했는지 확인
3. 인터넷 연결 상태 확인
4. 그래도 안 되면 관리자에게 문의

### Q2: 내 이름이 선수 목록에 없어요
**A:** 관리자에게 선수 등록을 요청해 주세요:
- 이름, 포지션, 연락처 정보 제공
- 관리자가 등록 후 바로 사용 가능합니다

### Q3: 모바일에서 달력이 작게 보여요
**A:** 모바일에 최적화된 기능입니다:
- 경기를 터치하면 출석 관리로 바로 이동
- 하단의 경기 목록에서도 같은 기능 이용 가능
- 세로 모드 사용을 권장합니다

### Q4: 과거 경기 출석을 변경할 수 있나요?
**A:** 일반적으로 과거 경기는 변경할 수 없습니다:
- 통계의 정확성을 위해 제한됩니다
- 특별한 경우 관리자에게 문의해 주세요

### Q5: 출석률은 어떻게 계산되나요?
**A:** 출석률 계산 방법:
```
출석률 = (참석한 경기 수 / 전체 경기 수) × 100%
```
- "미정" 상태는 "불참"으로 계산됩니다
- 정확한 통계를 위해 꼭 상태를 업데이트해 주세요

### Q6: 팀 소식은 누가 작성하나요?
**A:** 팀 관리자가 작성합니다:
- 중요한 공지사항
- 경기 일정 변경
- 팀 이벤트 소식 등

## 🆘 도움이 필요할 때

### 기술 문제
- 페이지가 로드되지 않을 때: 브라우저 새로고침
- 버튼이 작동하지 않을 때: 페이지 새로고침 후 재시도
- 모바일에서 문제 발생: 브라우저 업데이트 확인

### 계정/정보 문제
- 내 정보 수정: 관리자에게 요청
- 선수 등록 문제: 관리자에게 문의
- 출석 데이터 오류: 관리자에게 신고

### 연락처
팀 관리자에게 문의하거나 팀 단체 채팅방을 이용해 주세요.

## 🎯 효과적인 사용 방법

### 출석 관리 베스트 프랙티스
1. **정기적인 확인**: 일주일에 한 번씩 출석 상태 확인
2. **빠른 응답**: 경기 일정이 나오면 가능한 빨리 출석 상태 입력
3. **상태 업데이트**: 상황이 변하면 즉시 상태 변경
4. **팀 배려**: 다른 팀원들이 계획을 세울 수 있도록 정확한 정보 제공

### 팀 소통 향상
1. **소식 확인**: 정기적으로 팀 소식 페이지 확인
2. **사진 공유**: 갤러리를 통해 팀의 추억 공유
3. **통계 동기부여**: 출석률 통계를 통해 팀 참여 의식 향상
